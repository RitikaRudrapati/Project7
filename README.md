This project is a Python-based translator that converts Hack Virtual Machine (VM) code into Hack Assembly code (.asm).



It allows for basic operations like push/pop, arithmetic commands, and logical operations.

This program is specific to Hack Assembly



Features:



Push/Pop:


1- constant
2- local
3- argument
4- this
5- that
6- temp
7- pointer
8- static



Arithmetic Operations:



1- add
2- sub
3- and
4- or



Logical:



1- eq
2- gt
3- lt





How to Use it:

create a .vm file
edit the readFile() call in project7.py with the correctfile path
run the python script
the translated assembly code will be written to an .asm file
